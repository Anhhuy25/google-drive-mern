* What is "repo"?
- Có thể xem là source code của toàn bộ dự án.

* What is "commit"?
- Giúp dev biết là có code bị sửa, thêm file, xóa file. Được lưu vào repo và từ đó sẽ biết ai là người sửa

* What is "git init"?
- Tạo 1 Git repo trong máy 

* What is "git clone"?
- Lấy code từ trên Internet về

* What is "git pull"?
- Đồng bộ code từ trên Internet về máy

* What is "git add" or "git add ."?
- Thêm, sửa, xóa repo trong máy

* What is "git push"?
- Đồng bộ code từ trên máy lên Internet

* What is "git status"?
- Xem trạng thái các file trong máy

* What is "git log"?
- Xem trạng thái các file trong repo

* What is "git branch" + branch_name?
- Thêm một branch mới ngoài branch master sẵn có

* What is "git checkout"?
- Chuyển đổi giữa các branch
